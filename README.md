<h2 id="install"><strong>ğŸ’¾ Installation</strong></h2>

### **From scratch:**

``` bash
npm create @xl-soft/ui@latest
```

### **Import in existing project:**

``` bash
npm i @xl-soft/ui@latest
```

> Then add to your tsconfig.json

```json
{
    "compilerOptions": {
        "moduleResolution": "Bundler"
    },
}
```

### You are ready to go!
